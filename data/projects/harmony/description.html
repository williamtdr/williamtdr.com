Allows for control of a personalized dashboard via voice with the Amazon Echo. Integrates with Google services and provides a variety of functions useful with a screen, like looking up YouTube videos, adding calendar events and reminders, and searching Google.

Winner of Top Overall for CodeDay Spring 2016. Built using node, express, and the googleapis package. Backgrounds are automatically pulled from konachan using a custom API wrapper, from the `nobody scenic` tags.
